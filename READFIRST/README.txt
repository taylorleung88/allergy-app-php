To run the application, please download XAMPP if you have not already. Copy the entire allergy folder into htdocs section of XAMPP directory. Inside the READFIRST folder, you will find a users SQL file. Follow this link to import the database:
https://mediatemple.net/community/products/dv/204403864/export-and-import-mysql-databases

Once done, start XAMPP and begin viewing login.php in localhost. From this page, the you are free to use the web application. Also in this folder is the final paper.
